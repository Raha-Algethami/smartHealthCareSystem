#  Smart Health Platform – Vision 2030

A healthcare management system developed in **Java** using **Object-Oriented Programming (OOP)** principles. The project is inspired by **Saudi health apps** and demonstrates how software can support digital healthcare services.

##  Project description

This System was done as part of my Computer Engineering projects to strengthen my understanding of Java programming and object-oriented programming.


##  Features

- 👨‍⚕️ Doctor management
- 🧑‍🤝‍🧑 Patient management
- 📅 Appointment scheduling
- 🔔 Appointment reminders
- 💊 Electronic prescription generation
- 💰 Appointment fee calculator
- 🖥️ Console-based healthcare simulation
- 🌐 Responsive healthcare website

##  Object-Oriented Programming Concepts

### Encapsulation
- Private attributes with controlled access through constructors and methods.

### Inheritance
- `Doctor` and `Patient` inherit from the `User` class.

### Polymorphism
- Method overriding using `displayInfo()`.
- Runtime polymorphism through the `Calculator` reference.

### Abstraction
- `Calculator` is implemented as an abstract class.

### Composition
- `HealthcareSystem` contains `Doctor` and `Patient` objects.
